Welcome to the official jOOPL Toolbox API reference!

jOOPL Toolbox is an object-oriented framework for JavaScript and its goal is exposing base class library providing reusable and very generic solutions that seamlessly integrates with other existing libraries.

In order to leverage full object-oriented programming, jOOPL Toolbox is built on top of the tiny but powerful jOOPL library that can be found on http://joopl.codeplex.com.

Keyboard Shortcuts

  • Press s to focus the API search box.

  • Use Up and Down to select classes, modules, and search results.

  • With the API search box or sidebar focused, use ⌘-Left or ⌘-Right to switch sidebar tabs.

  • With the API search box or sidebar focused, use Ctrl+Left and Ctrl+Right to switch sidebar tabs.
